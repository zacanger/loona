# loona

[WIP] CLI for checking Loona stats.

--------

## Usage

`npx loona`

If you're the sort of person who likes to install things globally,
`npm i -g loona` and then `loona`

### Options

* `-p [--playlist]`: Open a playlist of all Loona MVs with less than 1M views
